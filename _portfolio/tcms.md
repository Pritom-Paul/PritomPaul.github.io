## Overview

A full-stack internal web application built to streamline ATM service operations — tracking service requests from submission through resolution, with engineer assignment and reporting features.

## Key Contributions

- Contributed to a full-stack web system for tracking ATM service requests, engineer assignments, and call logs.
- Developed frontend UI components and dashboards for real-time service request monitoring and engineer-wise reporting.
- Designed a normalized relational database schema (ER and class diagrams) to support multi-entity call management.
