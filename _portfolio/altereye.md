**Status:** Production (AlterSense Limited)  
**Tools:** Python, OpenVINO, YOLO, GStreamer, MQTT, Redis

## Overview

A real-time multi-camera computer vision system built for industrial environments — detecting anomalies, monitoring activities, and delivering low-latency analytics to dashboards.

## Key Contributions

- Worked on a real-time multi-camera vision system for industrial monitoring and anomaly detection.
- Developed pipelines for object detection and activity recognition using YOLO-based models.
- Processed multi-camera RTSP streams with GStreamer for scalable ingestion.
- Used MQTT for lightweight, real-time data transfer between system components.
- Stored intermediate results in Redis and persisted structured data in the database.
- Designed end-to-end flow for low-latency analytics and monitoring dashboards.
